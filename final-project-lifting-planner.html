Lifting Planner

by the Mongooses

(last updated Sprint 5 on 3/29/2021)


Description

This web application will include a user login interface and hold data given by the user in a database. Lifting Planner will take in the user's height, weight, and the max weight that

they can do for all core lifts and use that information to determine a workout plan that suits them. The workout plan will have the lifts that the user will be doing every day of the

week as well as how much weight is recommended for each lift. The purpose of this application is to have the user be able to easily access a workout plan that is right for them.


Features
  1. User login system held in database, which includes a login and signup page
  2. Form page for users to update their height, weight, and max weight that will also be sent to the database
  3. A "today's workout" page that shows the user what their workout is and updates every day
  4. A page displaying the workout plan for the week

Added if time allows

  1. Inputs for the user's weight that they are lifting for each workout
  2. Option to switch around workout plans for different days
  3. Option to adjust sets and reps in certain workouts

Languages and Platforms

  • GoLang will be used to write the client and server code
  • Microsoft Azure will be used to deploy the application

Technical features learned from class

  • Writing a GoLang application
  • Writing a GoLang server
  • Deploying a GoLang web application to Azure

Technical features learned outside of class

  • Creating a user login system
  • Uploading a JSON form to a database
  • Updating a page based on the date
